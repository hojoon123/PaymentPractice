# 쇼핑몰 & payment 프로젝트
- bootstrap5 기반 간단한 html & js 추후 React.js 마이그레이션 예정
- sqlite3 -> postgresSQL 마이그레이션 완료
- Django 5.1 Ver 기반 프로젝트 추후 DRF로 Restful API화 진행 예정
- settings.py를 살펴보고 .env의 변수 설정만으로 누구든지 가능함.
- 결제 시스템 portone API 사용
